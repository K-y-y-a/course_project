# Курсовой проект: Лента новостей

## Описание

Лента новостей с подгрузкой и комментариями.
Дизайн: [РИА Новости](https://ria.ru)

## Технологии

-   Vue.js
-   HTML5
-   JavaScript
-   TypeScript
-   CSS

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
